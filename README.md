# WattsonBot — Tesla Monitor for Telegram

**Real-time Tesla monitoring delivered directly to your Telegram.** Charging alerts, battery warnings, tire pressure, drive summaries and on-demand stats — all from a single bot.

🤖 **[@WattsonEVBot](https://t.me/WattsonEVBot)** &nbsp;·&nbsp; 🌐 **[wattsonbot.org](https://wattsonbot.org)**

---

## Features

| | |
|---|---|
| ⚡ **Charging alerts** | Start, stop, target level reached |
| 🔋 **Battery monitor** | Low-charge warning with configurable threshold |
| 🛞 **TPMS alerts** | Tire pressure drop notifications |
| 🗺️ **Drive events** | Trip start/end with location, distance, energy used |
| 📊 **Stats on demand** | Battery, range, temperature, charging history |
| 🔔 **Configurable** | Enable/disable each alert type from Telegram |
| 🌍 **Multilingual** | English and Russian |

---

## How It Works

```
1. Open @WattsonEVBot and press /start
2. Choose a plan — or start with a free 48-hour trial
3. Pay with ⭐ Telegram Stars or 💎 USDT (via @CryptoBot)
4. Send your Tesla refresh token — your private stack spins up in seconds
5. Receive real-time alerts and use /status, /battery, /location and more
```

Your data runs in an **isolated Docker stack** (TeslaMate + PostgreSQL + bot) — no shared infrastructure, no data mixing between users.

## Commands

| Command | Description |
|---------|-------------|
| `/status` | Car state, battery, range |
| `/battery` | Charging details |
| `/location` | Current location on map |
| `/trip` | Last trip summary |
| `/charging` | Active charging session |
| `/stats` | Today / 7 / 30 day statistics |
| `/tpms` | Tire pressure |
| `/climate` | Interior & exterior temperatures |
| `/billing` | Subscription management |
| `/reconnect` | Restart your bot instance |
